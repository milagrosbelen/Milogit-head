# MILOGIT SYSTEM

Sistema personal exclusivo de mentalidad y riqueza.

## Instalación

1. Abre una terminal en la carpeta del proyecto
2. Ejecuta los siguientes comandos:

```bash
npm install
npm run dev
```

3. Abre tu navegador en `http://localhost:5173`
